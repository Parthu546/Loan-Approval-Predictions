# Loan Approval Prediction System

## Overview

A Machine Learning project that predicts whether a loan application is likely to be approved based on applicant details such as income, credit history, education, employment status, and loan amount.

## Features

* Loan Approval Prediction
* Approval Probability Score
* EMI Calculator
* Interactive Streamlit Web Application
* User-Friendly Interface

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Streamlit
* Machine Learning

## Dataset

The project uses a loan approval dataset containing applicant information and loan status records.

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py
